# 对话管理功能使用指南

## 1. 功能概述

对话管理功能是智能家居智能体系统的核心功能之一，允许用户创建、管理和删除多个对话线程，实现多任务并行处理。

### 主要功能
- **多对话管理**：支持创建多个独立的对话线程
- **对话持久化**：对话历史自动保存，重启后仍可访问
- **对话命名**：支持自定义对话标题
- **对话删除**：可以删除不需要的对话记录
- **上下文隔离**：每个对话拥有独立的上下文，互不干扰

## 2. 系统架构

### 核心组件
- **SessionManager**：会话管理器，负责对话的创建、管理和持久化
- **API接口**：提供对话管理的RESTful API
- **前端界面**：用户交互界面，显示对话列表和聊天内容

### 数据存储
- **chats.json**：对话注册表，存储所有对话的基本信息
- **sessions/目录**：存储每个对话的详细历史记录

## 3. 使用方法

### 3.1 创建新对话
1. 在左侧对话列表上方，点击"+ 新建"按钮
2. 系统会创建一个名为"新对话"的新对话线程
3. 您可以立即开始与智能体对话

### 3.2 重命名对话
1. 方法一：点击对话列表中对应对话的"重命名"按钮
2. 方法二：在对话窗口顶部，点击"重命名"按钮
3. 在弹出的对话框中输入新的对话名称
4. 点击"确定"保存更改

### 3.3 删除对话
1. 方法一：点击对话列表中对应对话的"删除"按钮
2. 方法二：在对话窗口顶部，点击"删除"按钮
3. 在弹出的确认对话框中点击"确定"删除对话

### 3.4 切换对话
1. 在左侧对话列表中，点击要切换的对话
2. 系统会自动加载该对话的历史记录
3. 您可以继续与智能体在该对话中交流

### 3.5 对话历史管理
- 系统会自动保存所有对话历史
- 重启系统后，对话历史会自动恢复
- 对话历史会限制在最近20轮对话，以优化性能

## 4. API接口

### 4.1 获取对话列表
```http
GET /api/chats
```

**响应示例**：
```json
{
  "success": true,
  "chats": [
    {
      "channel": "Yueyue",
      "created_at": "2026-03-20T10:00:00.000Z",
      "id": "uuid",
      "meta": {},
      "name": "新对话",
      "session_id": "1773973885000",
      "updated_at": "2026-03-20T10:00:00.000Z",
      "user_id": "default"
    }
  ],
  "timestamp": "2026-03-20T10:00:00.000Z"
}
```

### 4.2 创建新对话
```http
POST /api/chats
Content-Type: application/json

{
  "name": "新对话",
  "user_id": "default"
}
```

**响应示例**：
```json
{
  "success": true,
  "chat": {
    "channel": "Yueyue",
    "created_at": "2026-03-20T10:00:00.000Z",
    "id": "uuid",
    "meta": {},
    "name": "新对话",
    "session_id": "1773973885000",
    "updated_at": "2026-03-20T10:00:00.000Z",
    "user_id": "default"
  },
  "timestamp": "2026-03-20T10:00:00.000Z"
}
```

### 4.3 更新对话名称
```http
PUT /api/chats/{session_id}
Content-Type: application/json

{
  "name": "新的对话名称"
}
```

**响应示例**：
```json
{

  "chat": {
    "channel": "Yueyue",
    "created_at": "2026-03-20T10:00:00.000Z",
    "id": "uuid",
    "meta": {},

  },
  "timestamp": "2026-03-20T10:00:00.000Z"
```

### 4.4 删除对话
```http
DELETE /api/chats/{session_id}
```

**响应示例**：
```json
{
  "success": true,
  "message": "对话删除成功",
  "timestamp": "2026-03-20T10:00:00.000Z"
}
```

### 4.5 获取对话历史
```http
GET /api/chats/{session_id}/history
```

**响应示例**：
```json
{
  "success": true,
  "history": [

    {"assistant": "你好！有什么我可以帮你的吗？"}

```

## 5. 部署指南

### 5.1 系统要求
- Python 3.11+
- FastAPI
- Uvicorn
- 七牛云AI推理API密钥

### 5.2 安装步骤
1. 克隆代码库
2. 安装依赖：`pip install -r requirements.txt`
3. 配置环境变量：
   - `QINIU_API_KEY`：七牛云AI推理API密钥
4. 启动服务：`python app.py`

### 5.3 数据存储
- 对话数据存储在 `~/.copaw/` 目录
- `chats.json`：对话注册表
- `sessions/`：对话历史存储
- `memory/`：全局记忆文件

### 5.4 性能优化
- 对话历史限制为最近20轮，以减少内存使用
- 会话数据定期持久化，确保数据安全
- 上下文隔离，避免不同对话之间的干扰

## 6. 故障排除

### 6.1 常见问题
1. **对话历史丢失**：检查 `~/.copaw/sessions/` 目录是否存在
2. **API调用失败**：检查七牛云API密钥是否正确配置
3. **前端界面不显示对话列表**：检查网络连接和API响应

### 6.2 日志查看
- 系统日志输出到控制台
- 错误信息会在前端界面显示
