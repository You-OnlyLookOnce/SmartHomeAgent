## ⚙️ 技术栈概览

- **底层框架**: ChatDev-inspired Multi-Agent System
- **Agent 数量**: 9 个专业智能体协同工作
- **记忆系统**: LanceDB + 三级记忆管理（短期/中期/长期）
- **LLM Provider**: 七牛云 Qwen 系列模型
- **硬件接口**: STM32 (C++) + Python API 层
- **人格注入**: Personality Layer (第 4 层)
