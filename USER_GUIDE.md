# 智能家居智能体用户指南

## 系统介绍

智能家居智能体是一个基于前沿AI架构的智能家居控制中心，能够通过自然语言交互控制家居设备、管理任务、提供个性化服务。系统采用三层隔离架构，集成了拟人化ReAct回复机制、混合检索系统和Gateway崩溃自修复机制，为用户提供智能、稳定、人性化的家居控制体验。

## 主要功能

### 1. 设备控制

- **灯光控制**：打开/关闭灯光，调节亮度

### 2. 任务管理

- **创建提醒**：设置一次性或重复提醒
- **安排任务**：创建和管理任务
- **发送通知**：发送系统通知

### 3. 智能交互

- **自然语言对话**：通过自然语言与智能体交流
- **拟人化回复**：智能体以温暖、专业的语气回复
- **情境感知**：根据时间、场景调整回复风格

### 4. 记忆系统

- **保存偏好**：记住用户的偏好设置
- **回忆记忆**：检索历史交互和偏好
- **记忆蒸馏**：自动分析和优化记忆

## 使用方法

### 1. 启动系统

1. 按照部署指南启动后端和前端服务
2. 在浏览器中访问系统界面

### 2. 与智能体对话

1. 在输入框中输入您的指令，例如：
   - "开灯"
   - "明天早上8点提醒我开会"
   - "室内温度是多少"
2. 点击"发送"按钮或按回车键
3. 智能体会分析您的指令并执行相应操作

### 3. 控制设备

1. 在设备控制面板中，点击相应设备的控制按钮
2. 对于灯光，可以调节亮度滑块

### 4. 管理任务

1. 通过对话创建提醒："提醒我明天下午3点取快递"
2. 通过对话安排任务："安排后天完成项目文档"

## 常见问题

### 1. 智能体没有响应

- 检查网络连接是否正常
- 检查后端服务是否正在运行
- 尝试刷新页面

### 2. 设备控制失败

- 检查设备是否在线
- 检查设备ID是否正确
- 尝试重新发送指令

### 3. 提醒没有触发

- 检查提醒时间是否设置正确
- 确保系统处于运行状态
- 检查系统时间是否准确

### 4. 记忆功能不工作

- 检查数据存储路径是否正确
- 确保系统有写入权限
- 尝试重新启动系统

## 高级功能

### 1. 自定义偏好

您可以通过对话设置个人偏好，例如：
- "我喜欢卧室温度保持在24度"
- "我希望晚上10点后自动关灯"

### 2. 场景模式

您可以创建和激活场景模式，例如：
- "开启电影模式"（关闭主灯，开启氛围灯）
- "开启睡眠模式"（关闭所有灯）

### 3. 语音控制

系统支持语音控制功能，您可以通过语音指令控制设备和管理任务。

## 故障排除

### 1. 系统崩溃

如果系统崩溃，Gateway守护者会自动尝试恢复系统。如果恢复失败，请检查日志文件并联系系统管理员。

### 2. 性能问题

如果系统响应缓慢：
- 关闭不必要的应用程序
- 增加系统内存
- 优化网络连接

### 3. 数据丢失

定期备份数据目录，以防止数据丢失。

## 联系方式

如果您在使用过程中遇到问题，请联系系统管理员或查看项目文档。
